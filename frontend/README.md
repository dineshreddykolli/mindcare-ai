# MindCare AI - Mental Health Triage Platform

A full-stack mental health intake and triage platform built with React, FastAPI, and PostgreSQL.

## Features
- AI-powered patient intake chat (Claude/Anthropic)
- Automated risk scoring and triage
- Admin dashboard with real-time clinic operations
- Therapist portal with caseload management
- High-risk patient alerts

## Tech Stack
- **Frontend:** React 18, Vite, Tailwind CSS, Recharts
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL
- **AI:** Anthropic Claude API
